
# Shared Types Package

TypeScript type definitions and DTOs for SHALKAAR platform.

## Usage

``` typescript
import { User, Product, Order, UserRole } from '@shalkaarnext/shared-types';

``` text

## Structure

- `enums.ts` - Enums for roles, statuses, types

- `types.ts` - Domain entity types

- `dtos.ts` - Request/Response DTOs

- `index.ts` - Exports and constants

## Build

``` bash
pnpm build

``` text
